# THREAD: Thinking Deeper with Recursive Spawning

## Dependencies
For ALFWorld and WebShop, install and set up the environments from the source repositories [alfworld](https://github.com/alfworld/alfworld), [webshop](https://github.com/princeton-nlp/WebShop). For TextCraft, please download the [crafting recipes](https://github.com/InventivetalentDev/minecraft-assets/tree/1.16.5/data/minecraft/recipes) and store them in `textcraft/textcraft/recipes`.
